Info Lowongan Pekerjaan



Desain Grafis


Kami sedang mencari desainer grafis yang kreatif dan berbakat untuk bergabung dengan tim kami. Tanggung jawab pekerjaan termasuk:


  • Merancang materi promosi dan iklan
  • Mengembangkan desain grafis untuk situs web dan media sosial
  • Menghasilkan konten visual untuk kampanye pemasaran

Jika Anda memiliki keterampilan desain grafis yang kuat dan siap untuk tantangan baru, silakan kirimkan portofolio Anda dan CV ke email kami.





Pengembang Web (Web Developer)


Kami sedang mencari pengembang web yang berpengalaman untuk bergabung dengan tim kami. Tanggung jawab pekerjaan termasuk:


  • Mengembangkan dan memelihara situs web menggunakan bahasa pemrograman seperti HTML, CSS, dan JavaScript
  • Mendesain tata letak dan fungsionalitas situs web
  • Mengoptimalkan kinerja situs web dan memastikan ketersediaan dan keamanannya

Jika Anda memiliki keterampilan pengembangan web yang kuat dan pengalaman yang relevan, silakan kirimkan portofolio dan CV Anda ke email kami.
